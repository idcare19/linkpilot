# LinkPilot
LinkPilot is a production ready url shortener Saas built with next js , django Rest Framework , and PostgreSQL.

# Goal

the goal is this project is to build a production ready url shortener Saas.

# Tech Stack

- Frontend: Next js
- Backend: django Rest Framework
- Database: PostgreSQL
- Authentication: JWT
- Deployment: vercel,render

# Planned Features

- User Registration and Login
- URL Shortening
- URL Tracking
- URL Analytics
- URL Expiration
- URL Customization
- URL Sharing
- click Tracking
- qr Code Generation


# Status
current status is development stage in progress.
